Run Correct Horse locally on your computer

There is an electron app available here:

correcthorse.asar.

Simple install

You will need to download Electron. Go to

https://github.com/electron/electron/releases/tag/v1.3.3

and download the Electorn release according to your system, e.g. electron-v1.3.3-win32-x64.zip for a 64 bit Windows system.

Then rename correcthorse.asar to app.asar and move it to the resouce directory of Electron and then start Electron (on a Mac you will have to move it to the Electron.app/Contents/Resources directory).
